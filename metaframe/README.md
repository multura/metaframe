# metaframe
Интерфейс в стиле Nest Hub (ну почти в стиле) для цифровой фоторамки.


## Функции

- Плавный слайд-шоу
- Погода
- Время (зависит от времени на вашем устройстве)




## Установка

Проверьте установку php и git (если есть):
```zsh
php -v
git --version
```
_Если нет, то следуйте инструкциям по установке для вашей ОС_

Скопируйте репозиторий:
```zsh
git clone https://github.com/multura/metaframe && cd metaframe
```
Запустите metaframe (проверив, занят ли порт):
```zsh
php -S localhost:1515
```
После чего ваша копия metaframe будет доступна по адресу http://localhost:1515/
## Конфигурация
### Настройка погоды
Перейдите к файлу weather.php и найдите поля
```php
$lat = 44.039815;
$lon = 43.070427;
```
Замените их на десятичные координаты вашего города
### Замена картинок в слайдшоу
Перейдите к директории photos\
Далее удалите ненужные вами фото, а после закиньте ваши (желательно .png или .jpeg)
